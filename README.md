## Installation

```bash
npm i svelte-code-copy
```

## Usage

`pre` is optional.
`<CodeCopy>` copies the `innerText` of `<code>`

```svelte
<script>
  import { CodeCopy } from 'svelte-code-copy'
</script>

<CodeCopy>
  <pre>
    <code>
      <!-- Your code block -->
    </code>
  </pre>
</CodeCopy>
```
